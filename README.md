# Modern Card Carousel Demo

一个卡片轮播展示页面，使用 React + TypeScript + Tailwind CSS + Swiper 构建。

## 🌟 特性

- 响应式设计，完美适配移动端/平板/桌面端
- 3D 覆盖流轮播效果
- 自动播放和循环播放
- 交互动画和过渡效果
- TypeScript 类型支持
- Tailwind CSS 样式管理

## 🛠️ 技术栈

- React 18
- TypeScript
- Tailwind CSS
- Swiper 10
- Vite

## 🚀 如何使用

1. 克隆项目到本地

```bash
git clone 
```

2. 安装依赖

```bash
cd  ...
npm install
```

3. 启动开发服务器

```bash
npm run dev
```

4. 打开浏览器访问 `http://localhost:5173` 查看效果

## 📝 注意事项

- 请确保你的开发环境已经安装了 Node.js 和 npm
